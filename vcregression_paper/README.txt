This folder contains the code and data necessary for reproducing all Figures in the main text and SI Appendix for the paper "Higher-order epistasis and phenotypic prediction". 

"Conceptual_figure.nb": Mathematica notebook for making Figures 1, S1, S2. 

"Smn1" folder: Code and data for making Figures 4, 5, 6, S7, S8, S9, S10, S11, S13.

"GB1" folder: Code and data for making Figures 3, S13. 

"Simulations" folder: Code for making Figures 2, S3, S4, S5, S6
